optimal:  Sieve of Eratosthenes
```python
class Solution:
# O(N*lglgN) + O(N)
    def countPrimes(self, n: int) -> int:
        cnt = 0
        if n < 2:
            return cnt 
        primes = [1]*(n)
        primes[0] = primes[1] = 0
        for i in range(2, int(sqrt(n))+1):
            if primes[i] == 1:
                for j in range(i*i,n,i):
                    primes[j] = 0
        return sum(primes)

```
